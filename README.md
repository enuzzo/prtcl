# PRTCL

Free, open-source web tool for creating, customizing, and exporting GPU-accelerated particle effects. Pick an effect, tweak it in real-time, export a self-contained snippet for any website. No accounts, no backend — runs entirely in the browser.

**[prtcl.es](https://prtcl.es)**

## Stack

Vite 6 · React 19 · TypeScript · React Three Fiber · Three.js · Zustand · Tweakpane · Tailwind CSS 4

## Getting started

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # Production build → dist/
```
