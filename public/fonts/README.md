# Fonts

## ProggyClean Nerd Font Mono

- **File**: `ProggyCleanNerdFontMono-Regular.ttf`
- **Author**: Tristan Grimmer (original), Nerd Fonts (patched v3.4.0)
- **License**: MIT — see `ProggyClean-Licence.txt`
- **Source**: https://github.com/bluescan/proggyfonts
- **Nerd Fonts**: https://github.com/ryanoasis/nerd-fonts
- **Usage**: Primary monospace UI font for the PRTCL editor (effect browser, control panel, status bar, code displays)
- **Fallback chain**: ProggyClean → JetBrains Mono → system monospace

### Why ProggyClean?

Pixel-perfect bitmap-style font optimized for small sizes. Sharp, readable, and distinctive — gives PRTCL a retro-technical aesthetic that complements the particle engine theme. The Nerd Font Mono variant includes coding ligatures and icon glyphs while maintaining strict monospace metrics.
